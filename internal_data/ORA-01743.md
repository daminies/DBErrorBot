# ORA-01743: only pure functions can be indexed

대상 서버: 2C-GROOT-DB
장애 코드: ORA-01743
오류 메시지: only pure functions can be indexed

서비스 영향도: 서비스 영향도 없음

발생 이력: 2회
1. 250501 11:12:40
2. 250611 15:41:02

원인:
- 12c 이상의 oracle db에서 비결정적 함수를 이용해 function based index를 생성하려고 할 경우 발생
- 비결정적 함수(EX. REGEXP_REPLACE)는 가상 열이나 함수 기반 인덱스에 넣을 수 없음
- 항상 같은 입력값에 대해 같은 결과를 반환해야 함

해결 방법:
1. DETERMINISTIC함수를 생성하여 함수 기반 인덱스 생성
