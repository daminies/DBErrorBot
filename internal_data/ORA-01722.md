# ORA-01722:  Cause: java.sql.SQLException : 수치 부적합


대상 서버: 2C-GROOT-DB
장애 코드: ORA-01722
오류 메시지: 수치 부적합.

서비스 영향도: 서비스 영향도 없음

발생 이력: 2회
1. 250501 11:12:40
2. 250611 15:41:02

원인:
- DATA PUMP시 존재하지 ㅇㅇ않는 DIRECTORY 명을 사용할 경우 발생
-

해결 방법:
1. `SELECT * FROM DBA_DIRECTORIES;`
2. `CREATE OR REPLACE DIRECTORY my_dir AS '/path/to/directory';`
