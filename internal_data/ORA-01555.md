*ORA-01555: Snapshot Too Old*


▶ *서비스 영향도( 없음)*
```
해당 오류는 특정 SQL 문장에 한해 정상적으로 완료되지 못하고 중단되며, 기타 서비스에는 직접적인 영향 없음
```

▶ *발생 이력 ( 2회)*
```
250501 11:12:40
250611 15:41:02
```

▶ *원인*
```
1. Undo 영역이 너무 작아 데이터 보존 실패
2. 장기 실행 쿼리로 인한 Undo 손실
```

▶ *해결 방법(ORA-01555.sql 스크립트 참고)*
```
1. 쿼리 튜닝을 통한 실행 시간 단축
2. UNDO_RETENTION 파라미터 증가 (`ALTER SYSTEM SET UNDO_RETENTION = 900;`)
3. Undo Tablespace 확장
```
