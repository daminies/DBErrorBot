*ORA-00060*

*▶ 서비스 영향도*
서비스 영향도 없음



*▶ 오류 원인*
ORA-00060 deadlock detected while waiting for resource
서로 다른 세션이 서로 Lock 을 걸고 서로 상대가 풀어주기만을 무한히 기다리는 경우 발생

A세션 : `UPDATE DAMIN.STUDENT SET NAME='TEST1' WHERE ID=1;` -> LOCK  걸린 상태
B세션 : `UPDATE DAMIN.STUDENT SET NAME='TEST2' WHERE ID=2;` -> LOCK  걸린 상태

-- COMMIT하지 않은 상태에서 아래와 같이 서로의 블록을 변경하려고 할 때 발생

A세션 : `UPDATE DAMIN.STUDENT SET NAME='TEST2' WHERE ID=2;` -> LOCK이 걸렸기 때문에 무한대기
B세션 : `UPDATE DAMIN.STUDENT SET NAME='TEST1' WHERE ID=1;` -> LOCK이 걸렸기 때문에 무한대기

*▶ 해결방안*
잠금 또는 호출 순서 변경

호출을 순차적으로 하여 교착 상태 발생 가능성 최소화
